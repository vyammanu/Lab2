Lab 2: Insertion Sort in C

This C program demonstrates the Insertion Sort algorithm to sort an array of integers.


Compiling the Program

To compile the `lab2.c` source file into an executable, follow these steps:

1. Open a terminal or command prompt.
2. Navigate to the directory containing the `lab2.c` source file.
3. Compile the program using a C compiler (e.g., GCC):

   gcc -o lab2 lab2.c

Running the program

./lab2
